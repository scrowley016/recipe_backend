POST /api/auth/register

Create a new user with a username and password. Password must be longer than 6 characters.

fetch("https://fsa-recipe.up.railway.app/api/auth/register", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    username: "johndoe",
    password: "securepassword"
  })
});

Returns:

{
  "token": "jwt_token_here"
  "username" : "username here"
}

POST /api/auth/login

Log in with a username and password to receive a JWT token user on future authenticated API calls

fetch("https://fsa-recipe.up.railway.app/api/auth/login", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    username: "janedoe",
    password: "securepassword"
  })
});

Returns:

{
  "token": "jwt_token_here"
  "username" : "username here"
}

GET /api/auth/me

Returns info about the currently logged-in user.

Header: Authorization: Bearer your_token_here

fetch("https://fsa-recipe.up.railway.app/api/auth/me", {
  headers: {
    Authorization: "Bearer your_token_here"
  }
});

Returns:

{
  "id": 1,
  "username": "janedoe"
}

POST /api/user-recipes

Create a new custom recipe. Requires authentication.

Header: Authorization: Bearer your_token_here

            fetch("https://fsa-recipe.up.railway.app/api/recipe/user-recipes", {
              method: "POST",
              headers: {
                "Content-Type": "application/json",
                Authorization: "Bearer your_token_here"
              },
              body: JSON.stringify({
                strMeal: "Grandma's Apple Pie",
                strCategory: "Dessert",
                strArea: "American",
                strInstructions: "1. Peel apples... 2. Make crust...",
                strMealThumb: "https://example.com/images/applepie.jpg",
                strTags: "Pie,Dessert,Fruit",
                strYoutube: "https://youtube.com/applepie",
                ingredients: [
                  "2 apples",
                  "1 cup flour",
                  "1/2 cup sugar"
                ]
              })
            });

Returns:

            {
              "message": "Recipe created successfully!",
              "data": {
                "id": 1,
                "userId": 1,
                "strMeal": "Grandma's Apple Pie",
                ...
              }
            }

DELETE /api/user-recipes/:id

Delete a custom recipe by its ID. Only the user who created the recipe can delete it.

Header: Authorization: Bearer your_token_here

            fetch("https://fsa-recipe.up.railway.app/api/recipe/user-recipes/1", {
              method: "DELETE",
              headers: {
                Authorization: "Bearer your_token_here"
              }
            });

Returns:

            { "message": "Recipe deleted.", "id": 1 }
